CommsView

The CommsView is a view into a RobotCommunicator that other threads/objects can use to request that messages be sent.

RobotCommunicator

A RobotCommunicator is a complete Comms system which can be run. It has a CommsController that it owns and a driver that it owns for io purposes. The C type parameter is the type of the CommsController. The I type parameter is the type of the IoServerManager.
